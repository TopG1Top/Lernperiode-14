# Lernperiode 14
Semesterplanung – Advanced Webentwicklung & Ethical Hacking / Security

## Grobplanung Semester
In dieser Lernperiode fokussiere ich mich auf zwei fortgeschrittene Bereiche:  
(1) Webentwicklung mit Fokus auf Architektur, saubere Struktur, API-Handling und nachvollziehbare Datenflüsse. Ziel ist es, Projekte nicht nur funktional, sondern auch wartbar und klar aufgebaut umzusetzen.  
(2) Ethical Hacking & Security mit Fokus auf Analyse von Netzwerkverkehr, Verständnis von Requests sowie Erkennen von Schwachstellen. Dabei arbeite ich ausschliesslich im legalen Kontext und konzentriere mich auf Analyse und Absicherung.

Ziel ist, am Ende der Lernperiode messbare Resultate zu haben:  
- Eine erweiterte Web-App mit mehreren Features und sauberer Struktur  
- Vertieftes Verständnis von HTTP, API-Calls und Netzwerkverkehr  
- Praktische Analysen mit Wireshark (gezielte Filter, Interpretation von Traffic)  
- Dokumentierte Security-Erkenntnisse (Schwachstellen + Analyse-Beispiele)  
- Verbesserte Code-Qualität (Struktur, Naming, evtl. Testing)
